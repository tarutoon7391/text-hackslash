# ⚔️ テキストハックスラッシュ

> 文字を武器に、ダンジョンを攻略するブラウザRPGのランディングページ

## 🎮 ゲーム概要

**テキストハックスラッシュ**は、ブラウザだけで遊べるテキストベースのハック＆スラッシュRPGです。  
派手なグラフィックは不要。テキストで繰り広げるスピーディーなバトルと、深い装備・スキルシステムが特徴です。

### 主な特徴

- **テキスト式バトル** — サクサク進むテンポが最高の中毒性を生む
- **ランダム性が熱い** — D1〜D12の12種ダンジョン、ダイスの目でドロップが変わる
- **装備強化システム** — normal → rare → legend のレアリティと豊富な強化要素
- **レベルアップ・スキルツリー** — Lv50まで成長、SPでビルドをカスタマイズ


## 🎨 デザインについて

- **フォント**: Fredoka One（ディスプレイ）/ Boogaloo（サブ）/ Nunito（ボディ）
- **カラーパレット**: ホットピンク・オレンジ・イエロー・ライムグリーン・スカイブルー
- CSS カスタムプロパティで色を一元管理
- ページロード時のスタッガードリビールアニメーション
- スクロールトリガーによる要素の出現演出
